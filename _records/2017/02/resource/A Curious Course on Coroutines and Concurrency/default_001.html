[ David Beazley ]

Welcome! I am an independent software developer, teacher, and book author living in the city of Chicago. I primarily work on programming tools and teach programming courses for software developers, scientists, and engineers. I am the author of the Python Essential Reference (Addison-Wesley) and Python Cookbook, 3rd Ed. (O'Reilly Media). I also recently recorded The Python Programming Language" video training series for Pearson. Although Python is my current language of choice, I also have significant experience with systems programming in C, C++, and assembly language.
